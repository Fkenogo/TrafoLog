## Report 4 — District Asset Summary

**Format:** PDF  
**Filename:** `kVAssetTracker_DistrictSummary_YYYY-MM-DD.pdf`

A summary report grouped by district. For each district, show:

| Metric | Value |
|---|---|
| Total Transformers | count |
| Active | count |
| Faulty | count |
| Under Maintenance | count |
| Decommissioned | count |
| 11kV Network | count |
| 33kV Network | count |
| Open Faults | count |
| Overdue Inspection (90+ days) | count |

**PDF Layout:**
- Header: "kVAssetTracker — District Asset Summary", generated date, generated by user
- One section per district with the metrics table
- Page numbers in footer
- kVAssetTracker branding in header

Use `jspdf-autotable` for table generation.

## Export Buttons on Other Pages

**On Transformers List page:**
```
[Export ▾]
  → Export Current View (Excel)
  → Export Current View (CSV)
```
Exports only the currently filtered/visible data.

**On Manager Dashboard:**
```
[Export ▾]
  → Asset Register
  → Inspection Report
  → Fault Report
  → District Summary (PDF)
```
Each opens the Reports page pre-configured for that report type.

## RBAC

- Territory Manager: export scoped to their territory only (enforced server-side)
- Super Admin and Engineer: full export access
- Viewer: read-only export access

---

## Notes

> `jspdf-autotable` handles PDF table generation — do not build custom PDF table layouts.

> For exports with 10,000+ rows, process in chunks of 500 rows to avoid browser memory issues. Show a progress percentage.

> CSV format: use comma delimiter, UTF-8 encoding, include header row.
